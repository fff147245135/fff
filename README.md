# 毕业创作作品集网站

这是一个展示毕业创作作品的个人网站，包含五个主题：几何、城市、杯中世界、梦境、规则。

## 功能特点

- 🎨 现代化的响应式设计，适配各种设备
- 🖼️ 精美的图片画廊展示
- 🔍 点击图片可全屏查看
- ⌨️ 支持键盘导航（左右箭头键切换图片，ESC关闭）
- 📱 移动端友好的导航菜单
- ⚡ 图片懒加载，优化加载速度
- 🎭 流畅的动画效果

## 使用方法

### 方法一：直接打开（本地查看）

1. 确保所有文件都在同一个文件夹中
2. 双击 `index.html` 文件
3. 网站会在浏览器中打开

### 方法二：使用本地服务器（推荐）

#### 使用 Python（如果已安装）

在项目文件夹中打开终端/命令行，运行：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问：`http://localhost:8000`

#### 使用 Node.js（如果已安装）

安装 http-server：
```bash
npm install -g http-server
```

在项目文件夹中运行：
```bash
http-server
```

然后在浏览器中访问显示的地址（通常是 `http://localhost:8080`）

## 部署到网络

### 方法一：GitHub Pages（免费，推荐）

1. 在 GitHub 上创建一个新仓库
2. 将所有文件上传到仓库
3. 进入仓库设置（Settings）
4. 找到 "Pages" 选项
5. 选择主分支（main 或 master）作为源
6. 保存后，GitHub 会提供一个网址，例如：`https://你的用户名.github.io/仓库名`

### 方法二：Netlify（免费，简单）

1. 访问 [Netlify](https://www.netlify.com/)
2. 注册/登录账号
3. 将整个项目文件夹拖拽到 Netlify 的部署区域
4. 网站会自动部署并获得一个网址

### 方法三：Vercel（免费，快速）

1. 访问 [Vercel](https://vercel.com/)
2. 注册/登录账号
3. 导入项目（可以连接 GitHub 仓库或直接上传）
4. 自动部署完成

### 方法四：其他静态网站托管服务

- **Surge.sh** - 简单的命令行部署工具
- **Firebase Hosting** - Google 的托管服务
- **Cloudflare Pages** - Cloudflare 的免费托管服务

## 自定义修改

### 修改网站标题

编辑 `index.html` 文件，找到：
```html
<title>毕业创作作品集 - 个人网站</title>
```
修改为你想要的标题。

### 修改主题描述

在 `index.html` 中找到各个主题的 `section-description` 部分，修改描述文字。

### 添加新作品

1. 将新图片放入对应的主题文件夹
2. 在 `index.html` 中找到对应的主题区域
3. 复制一个 `gallery-item` 的代码块
4. 修改 `src` 属性为你的新图片路径

### 修改颜色主题

编辑 `styles.css` 文件，在文件开头的 `:root` 部分修改颜色变量：
```css
:root {
    --primary-color: #2c3e50;    /* 主色调 */
    --secondary-color: #3498db;  /* 次要颜色 */
    --accent-color: #e74c3c;     /* 强调色 */
    /* ... */
}
```

## 注意事项

1. **图片路径**：确保所有图片路径正确，图片文件名不要包含特殊字符
2. **文件大小**：建议图片文件不要过大（建议每张图片小于 2MB），以提升加载速度
3. **图片格式**：支持 JPG、PNG、WebP 等常见格式
4. **中文路径**：如果遇到中文路径问题，建议使用英文文件夹名

## 常见问题

### Q: 图片显示不出来？
A: 检查图片路径是否正确，确保图片文件确实存在于对应文件夹中。

### Q: 如何分享给他人？
A: 部署到 GitHub Pages、Netlify 等平台后，将网址分享给他人即可。

### Q: 可以添加更多主题吗？
A: 可以，按照现有主题的结构添加新的 section 即可。

### Q: 网站加载很慢？
A: 可以压缩图片大小，或使用图片懒加载功能（已内置）。
